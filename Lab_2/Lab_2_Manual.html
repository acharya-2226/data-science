Performing Statistical Analysis

1. Descriptive Statistics

Descriptive statistics summarize and describe the main features of a dataset.

We'll calculate:

  • Mean, median, mode
  • Standard deviation and variance

2. Inferential Statistics

Inferential statistics allow us to make predictions or inferences about a population based on a sample.

We'll perform:

  • Hypothesis testing using t-tests.
  • Correlation analysis.

2. Pandas: Data Manipulation

Pandas is a library for handling structured data. It provides:

  • DataFrame and Series structures for data handling.
  • Functions for data cleaning, filtering, and grouping.

Basic Operations

Below are examples to get started with Pandas.

In[21]:
import pandas as pd

# Creating a DataFrame
data = {'Name': ['Alice', 'Bob', 'Charlie'], 'Age': [25, 30, 35], 'Score': [85, 90, 95]}
df = pd.DataFrame(data)
print("DataFrame:\n", df)

# Basic operations
print("Mean age:", df['Age'].mean())
print("Data sorted by Score:\n", df.sort_values(by='Score', ascending=False))

3. Data Visualization: Matplotlib and Seaborn

Matplotlib and Seaborn are libraries for visualizing data through graphs and plots.

Basic Operations

Below are examples to create visualizations.

In[4]:
import matplotlib.pyplot as plt
import seaborn as sns

# Sample data
data = [12, 15, 14, 10, 18, 20, 15, 14]

# Line plot using Matplotlib
plt.plot(data, marker='o')
plt.title("Line Plot")
plt.xlabel("Index")
plt.ylabel("Value")
plt.show()

# Boxplot using Seaborn
sns.boxplot(data=data)
plt.title("Boxplot")
plt.show()

# Generate a histogram for random data of size 100 using Seaborn:

import seaborn as sns
import numpy as np
import matplotlib.pyplot as plt

# Generating random data
random_data = np.random.randn(100)

# Creating the histogram
sns.histplot(random_data, bins=10, kde=True, color='purple')
# In the context of Seaborn's histplot function, the kde parameter stands for Kernel Density Estimate.
# When you set kde=True, it adds a smooth curve over the histogram, which represents the estimated probability density function of the data.

# This can be particularly useful for visualizing the distribution of data in a more continuous manner,
# as opposed to the discrete bins of a histogram. The KDE curve helps to identify
# patterns and trends in the data that might not be immediately apparent from the histogram alone.
plt.title('Histogram of Random Data')
plt.xlabel('Value')
plt.ylabel('Frequency')
plt.show()

# Create a scatter plot with a regression line using Seaborn:

import seaborn as sns
import numpy as np
import matplotlib.pyplot as plt

# Generating sample data
np.random.seed(42)
x = np.random.rand(50) * 10  # Random x values
y = 2.5 * x + np.random.randn(50) * 5  # Linear relation with noise

# Creating the scatter plot with regression line
sns.regplot(x=x, y=y, scatter_kws={'color': 'blue'}, line_kws={'color': 'red'})
plt.title('Scatter Plot with Regression Line')
plt.xlabel('X Values')
plt.ylabel('Y Values')
plt.show()
